üöÄ How to Get Started

Follow these simple steps to run the project locally:

  • Install Node.js: Make sure you have Node.js version 22 or higher installed on your machine.
  • Clone the Project: Download or clone the mystery-box-p folder to your computer.
  • Install Dependencies: Open terminal in the project folder and run: npm install
  • Login to Reddit: Connect your Reddit account: npm run login
  • Start Development Server: Run the playtest server: npm run dev
$ cd mystery-box-p $ npm install $ npm run login $ npm run dev ‚úÖ Playtest ready!
